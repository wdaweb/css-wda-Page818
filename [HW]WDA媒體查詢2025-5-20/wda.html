HTML

HTML（超文字標記語言）是建立網頁的基礎語言，用來定義網頁的結構與內容，例如文字、圖片、連結等。它透過標籤（tag）讓瀏覽器知道如何呈現資訊。

CSS

CSS（層疊樣式表）用來控制 HTML 網頁的外觀與樣式，例如顏色、字型、排版與版面配置。它讓網頁更美觀且更易於設計與維護。

JavaScript

JavaScript（簡稱 JS）是一種用於網頁的程式語言，能讓網頁具有互動功能，例如按鈕點擊、表單驗證、動畫效果等，是前端開發的重要技術之一。

jQuery

jQuery（簡稱 jQ）是一個輕量級的 JavaScript 函式庫，簡化了操作 DOM、處理事件、動畫效果與 Ajax 的語法，讓網頁互動功能更容易開發與跨瀏覽器相容。

Bootstrap

Bootstrap 是一套開源的前端框架，提供現成的 CSS 和 JavaScript 元件，可快速建立響應式、跨裝置相容的美觀網頁，常用於佈局、按鈕、表單等設計。

Sass

SaaS（軟體即服務）是一種透過網路提供軟體應用的模式，用戶無需安裝，只需透過瀏覽器即可使用，方便更新與維護，常見於雲端辦公、電子郵件等服務。

Vue.js

Vue.js 是一款輕量且靈活的 JavaScript 前端框架，用於構建互動式用戶界面和單頁應用，具備簡單易學、組件化和響應式數據綁定的特點。
